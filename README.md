## 📖 About the Certification

This repository showcases my successful completion of the **IBM DB0111EN: Introduction to SQL** certification offered by **IBM through edX**.

The course provided a solid foundation in **Structured Query Language (SQL)** and relational database concepts, enabling me to write efficient queries for retrieving, analyzing, and managing data. It strengthened my understanding of SQL as a core skill for **Business Analytics**, **Data Analytics**, and **Business Intelligence**.

---

## 🎯 Learning Objectives

Throughout this certification, I learned how to:

- Understand relational database concepts
- Write SQL queries to retrieve and manipulate data
- Filter, sort, and organize datasets
- Perform aggregations using SQL functions
- Work with multiple tables using JOIN operations
- Use GROUP BY and HAVING clauses
- Write nested queries (Subqueries)
- Analyze structured datasets using SQL
- Apply SQL in real-world business scenarios
